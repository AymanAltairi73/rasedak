# توثيق الشاشات الجديدة

## نظرة عامة

تم إضافة ثلاث شاشات جديدة لتطبيق "رصيدك" بتصميم مطابق للصور المرفقة:

1. **New Order Screen** - شاشة إضافة طلب جديد
2. **Order Confirmation Screen** - شاشة تأكيد الطلب
3. **Profile Screen** - شاشة الملف الشخصي

---

## 1. New Order Screen (شاشة الطلب الجديد)

### المسار
`lib/screens/new_order_screen.dart`

### الوصف
شاشة لإضافة منتج سمك جديد للبيع مع جميع التفاصيل المطلوبة.

### المكونات الرئيسية

#### Header
- نص ترحيبي "مرحبا بك !"

#### العنوان والوصف
- عنوان: "عرض منتج جديد"
- وصف: "ادخل البيانات المطلوبة بالأسفل"

#### حقول الإدخال
1. **الاسم الكامل** - حقل نصي
2. **أرقام التواصل** - حقل رقمي
3. **الصنف** - قائمة منسدلة (نباض، هامور، تونة، ساردين)
4. **سعر الكيلو** - حقل رقمي
5. **الكمية المتوفرة** - حقل رقمي
6. **الوصف** - قائمة منسدلة (طازج، مجمد، مبرد)

#### إضافة الصور
- شبكة 2×2 (4 صور)
- كل مربع يحتوي على أيقونة + للإضافة
- عند الإضافة تظهر علامة ✓

#### زر التأكيد
- زر كبير في الأسفل
- عند الضغط: التحقق من البيانات → الانتقال لشاشة التأكيد

#### Bottom Navigation
- 3 أيقونات: Profile, Add (نشط), Home

### التنقل
```
NewOrderScreen → OrderConfirmationScreen
```

### التحقق من البيانات
- جميع الحقول مطلوبة
- التحقق يتم عند الضغط على زر التأكيد

---

## 2. Order Confirmation Screen (شاشة تأكيد الطلب)

### المسار
`lib/screens/order_confirmation_screen.dart`

### الوصف
شاشة لعرض ومراجعة تفاصيل الطلب قبل التأكيد النهائي.

### المكونات الرئيسية

#### Header
- نص ترحيبي "مرحبا بك !"

#### صورة البروفايل
- دائرة كبيرة بأيقونة المستخدم
- حجم: 100×100

#### معلومات البائع
- **أسم البائع**: مرعي فرج الزينوبي
- **الحساب الشخصي**: رابط
- **أرقام التواصل**: 3 أرقام مخفية جزئياً (77........)

#### صور السمك
- عنوان: "صور السمك"
- شبكة 2×2 داخل إطار أبيض
- عرض الصور المرفوعة

#### تفاصيل الطلب
- **نوع السمك**: التونة (أو القيمة المدخلة)
- **سعر الكيلو**: 8000 ريال
- **الكمية المتوفرة**: 3 كيلو
- **الوصف**: طازج

#### أزرار الإجراء
- **تأكيد**: زر أزرق - يؤكد الطلب ويعود للشاشة السابقة
- **الغاء**: زر أزرق - يلغي ويعود للشاشة السابقة

### التنقل
```
OrderConfirmationScreen → (Back to previous screen)
```

### الإجراءات
- **تأكيد**: عرض SnackBar نجاح + العودة
- **إلغاء**: العودة مباشرة

## 3. Profile Screen (شاشة الملف الشخصي)

### المسار
`lib/screens/profile_screen.dart`

### الوصف
شاشة عرض وإدارة معلومات المستخدم الشخصية.

### المكونات الرئيسية

#### App Bar
- العنوان: "الملف الشخصي" (في المنتصف)
- زر رجوع (يسار)
- زر للأمام (يمين)

#### معلومات المستخدم
- **صورة البروفايل**: دائرة 80×80
- **الاسم**: مرعي فرج الزينوبي
- **البريد**: Marae_frag_AL_enobe_gmail.com

#### قائمة الخيارات

##### 1. طلباتي
- **العنوان**: طلباتي
- **الوصف**: لديك : طلبات جاري تنفيذها
- **الأيقونة**: سهم أسود على اليسار

##### 2. أرقام التواصل
- **العنوان**: أرقام التواصل
- **الوصف**: يمكنك تعديل أرقام التواصل الخاصة بك
- **الأيقونة**: سهم أسود على اليسار

##### 3. الإعدادات
- **العنوان**: الإعدادات
- **الوصف**: يمكنك تعديل بياناتك الشخصية
- **الأيقونة**: سهم أسود على اليسار

#### Bottom Navigation
- 3 أيقونات: Profile (نشط), Add, Home

### التنقل
```
ProfileScreen → (TODO: Orders, Contact, Settings screens)
```
