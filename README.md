# رصيدك (Rasedak)

تطبيق Flutter لبيع الأسماك بواجهات عربية RTL مع تصميم عصري وسلس.

## المميزات

- ✅ واجهات عربية كاملة مع دعم RTL
- ✅ شاشة Splash مع انتقال سلس
- ✅ شاشات Onboarding (3 شاشات) قابلة للسحب
- ✅ نظام تسجيل دخول وإنشاء حساب
- ✅ تحقق من صحة البيانات (Email, Password)
- ✅ تصميم responsive يتكيف مع جميع الشاشات
- ✅ هيكلة نظيفة وبسيطة (Simple & Clean Architecture)

## التشغيل

### المتطلبات
- Flutter SDK (3.8.1 أو أحدث)
- Dart SDK

### خطوات التشغيل

1. **تثبيت الاعتماديات:**
```bash
flutter pub get
```

2. **تشغيل التطبيق:**
```bash
flutter run
```

3. **بناء التطبيق:**
```bash
# Android
flutter build apk

# iOS
flutter build ios
```

## الشاشات

### 1. Splash Screen
شاشة البداية مع الشعار وانتقال سلس (2 ثانية)

### 2. Onboarding Screens
3 شاشات تعريفية قابلة للسحب مع:
- مؤشر نقاط (dots indicator)
- زر تخطي
- زر التالي/ابدأ الآن

### 3. Login Screen
شاشة تسجيل الدخول تحتوي على:
- حقل البريد الإلكتروني
- حقل كلمة المرور مع إظهار/إخفاء
- خيار "تذكرني"
- رابط "نسيت كلمة المرور"
- أزرار تسجيل الدخول عبر وسائل التواصل

### 4. Signup Screen
شاشة إنشاء حساب جديد تحتوي على:
- حقل الاسم الكامل
- حقل البريد الإلكتروني
- حقل كلمة المرور
- حقل تأكيد كلمة المرور
- تحقق من صحة البيانات

### 5. Email Verify Screen
شاشة التحقق من البريد الإلكتروني:
- 5 حقول لإدخال رمز التحقق
- عدّاد زمني لإعادة الإرسال (60 ثانية)
- زر إعادة إرسال الرمز
- التحقق التلقائي عند إدخال الرمز كاملاً

### 6. Signup Successful Screen
شاشة نجاح إنشاء الحساب:
- رسوم متحركة للشعار
- رسالة نجاح
- زر للانتقال إلى التطبيق

### 7. Main/Home Screen
الشاشة الرئيسية تحتوي على:
- شريط علوي مع إشعارات
- فئات أفقية قابلة للتمرير
- شبكة منتجات الأسماك
- بطاقات منتجات مع صور وأسعار
- شريط تنقل سفلي

### 8. New Order Screen
شاشة إضافة طلب جديد:
- حقول إدخال البيانات (الاسم، الهاتف، الصنف، السعر، الكمية، الوصف)
- قوائم منسدلة للصنف والوصف
- شبكة لإضافة 4 صور للسمك
- زر تأكيد للانتقال لشاشة التأكيد

### 9. Order Confirmation Screen
شاشة تأكيد الطلب:
- صورة البروفايل
- معلومات البائع (الاسم، الحساب، أرقام التواصل)
- عرض صور السمك المرفوعة
- تفاصيل الطلب (النوع، السعر، الكمية، الوصف)
- أزرار تأكيد وإلغاء

### 10. Profile Screen
شاشة الملف الشخصي:
- معلومات المستخدم (الصورة، الاسم، البريد)
- قائمة الخيارات:
  - طلباتي
  - أرقام التواصل
  - الإعدادات
- شريط تنقل سفلي

## التحقق من البيانات

- **البريد الإلكتروني**: يجب أن يكون بصيغة صحيحة
- **كلمة المرور**: يجب أن تكون 8 أحرف على الأقل
- **تأكيد كلمة المرور**: يجب أن تطابق كلمة المرور
- **الحقول المطلوبة**: جميع الحقول إلزامية
